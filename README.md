# sales-data-analysis-python


Projeto em Python que demonstra ETL simples e análise de vendas com pandas e matplotlib.


## Conteúdo
- `src/etl.py` — funções para carregar e transformar dados
- `src/plots.py` — funções para gerar gráficos
- `src/main.py` — script principal
- `data/` — CSVs de exemplo (incluir)
- `outputs/` — gráficos e CSVs gerados
- `tests/test_etl.py` — testes unitários


## Como rodar
1. `python -m venv .venv` e `source .venv/bin/activate` (ou PowerShell no Windows)
2. `pip install -r requirements.txt`
3. `python src/main.py`


## Tecnologias
Python, pandas, matplotlib, pytest
